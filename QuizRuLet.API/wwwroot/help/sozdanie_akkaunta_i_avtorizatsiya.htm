4.1 Создание аккаунта и авторизация

Для использования онлайн-платформы пользователю требуется создать личный аккаунт. Если у пользователя не будет аккаунта он не сможет пользоваться функционалом онлайн-платформы.
Для начала регистрации пользователь должен перейти со страницы авторизации на страницу создания аккаунта, нажав на кнопку регистрации.

Кнопка для перехода к регистрации аккаунта

При отсутствии зарегистрированного аккаунта, требуется перейти на страницу регистрации.
Наверх

Поле для ввода Логина

В данном поле нужно ввести логин от аккаунта пользователя.
Наверх

Поле для ввода Пароля

В данном поле нужно ввести пароль от аккаунта пользователя.
Наверх

Кнопка Войти

После заполнения полей Логин, Пароль, требуется нажать на кнопку Войти для входа в пользовательский аккаунт.
Наверх

Поле для ввода Логина

Придумайте логин для своего аккаунта и заполните поле.
Наверх

Поле для ввода Пароля

Придумайте пароль для своего аккаунта и заполните поле.
Наверх

Поле для ввода Пароля

Повторите придуманный пароль.
Наверх

Кнопка Создать аккаунт

После заполнения всех полей, следует нажать на кнопку для создание аккаунта
Наверх

Кнопка для перехода на страницу авторизации

Если аккаунт уже зарегистрирован, следует перейти на страницу авторизации, нажав на кнопку.
