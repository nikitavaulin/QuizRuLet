4.6 Переход к изучению учебного модуля

Переход к изучению учебного модуля пользователь может осуществить несколькими вариантами.
Первый вариант для перехода к изучению учебного модуля:

Кнопка изучения учебного модуля

Кнопка для перехода к изучению учебного модуля
Наверх
Второй вариант для перехода к изучению учебного модуля:
Нажать на кнопку всплывающего меню.

Кнопка для вызова меню учебного модуля

Кнопка вызывает всплывающее меню учебного модуля
Наверх
Выбрать в меню соответствующую кнопку.

Меню учебного модуля

Меню с дополнительными операциями над учебным модулем.
Наверх

Кнопка изучения учебного модуля в меню

Кнопка для перехода к режиму изучения модуля.
Наверх
После выполнения любого из вариантов пользователю открывается страница с изучением учебного модуля.
Также пользователь может перейти к изучению модуля из режима редактирования.

Кнопка изучения учебного модуля

Кнопка для перехода к режиму изучения модуля из режима редактирования
Наверх
После нажатия кнопки пользователю открывается страница с изучением учебного модуля.
